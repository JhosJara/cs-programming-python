Overview
This repository contains programming assignments and projects from a 3-course Python sequence completed as part of my AAs in Computer Science. The coursework progressed from programming fundamentals through object-oriented design, data structures, and algorithm implementation.

Courses
CS1101 — Programming Fundamentals (Grade: A)
Introduction to Python programming: variables, data types, functions, conditionals, recursion, iteration, strings, lists, and dictionaries.
Key assignments:

Function design and modular programming
Data structure manipulation (lists, dictionaries)
Iterative and recursive problem solving


CS1102 — Programming 1 (Grade: A+)
Deeper Python programming: object-oriented concepts, file I/O, exception handling, and applied problem solving.
Key assignments:

OOP design and class implementation
File reading and data processing
Error handling and input validation


CS1103 — Programming 2 (Grade: A-)
Advanced programming concepts: algorithms, sorting, searching, complexity analysis, and applied data structures.
Key assignments:

Algorithm implementation and analysis
Complex data structure design
Software testing methodologies (equivalence partitioning, boundary-value analysis)
